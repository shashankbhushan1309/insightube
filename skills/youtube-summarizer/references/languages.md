# Supported Languages

This skill supports the following languages. Use the **Code** when passing arguments to scripts.

| Language | Code | Example Trigger Phrases |
|---|---|---|
| English | `en` | "/language english", "in english", "summarize in english" |
| Hindi | `hi` | "/language hindi", "in hindi", "हिंदी में", "hindi me" |
| Tamil | `ta` | "/language tamil", "in tamil", "தமிழில்" |
| Kannada | `kn` | "/language kannada", "in kannada", "ಕನ್ನಡದಲ್ಲಿ" |
| Telugu | `te` | "/language telugu", "in telugu", "తెలుగులో" |
| Marathi | `mr` | "/language marathi", "in marathi", "मराठीत" |

**Default Language:** If no language is specified and none is detected, default to `en` (English).
